# Supported Pipelines

This shared tooling repository supports Ghostcheese Games Unity asset pipelines for:

- UI Toolkit graphics
- 2D sprites
- 3D models
- textures/materials
- animation
- sound effects
- music
- voice-over
- VFX / particle / screen effects
- font / localized text-presentation

Localization-aware rules and fallback behavior are first-class concerns across applicable pipeline types.
