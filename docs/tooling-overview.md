# Tooling Overview

This repository is the shared implementation space for Ghostcheese Games Unity asset-pipeline tooling reused across multiple game repositories.

## Intended tooling categories

- package validation and completeness checks
- shared manifest/schema support
- extraction/export/import helpers
- handoff package support
- release-compatible validator/import contracts

## Design goals

- reusable across games
- deterministic in CI and local execution
- implementation-agnostic across tooling stacks/languages
- aligned with org governance

- shared asset-pipeline tooling governance
- game-repo tool consumption governance
